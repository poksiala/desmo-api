DESMOFYLAKAS(1) Manual Page

NAME

desmofylakas - Kubernetes-like orchestrator for FreeBSD Jails

SYNOPSIS

desmofylakas [OPTIONS] COMMAND [ARGS...]

DESCRIPTION

Desmofylakas is a orchestration tool designed specifically for FreeBSD Jails, inspired by the principles of Kubernetes. It provides a robust and flexible platform for managing isolated stateless applications in a scalable and efficient manner.

OPTIONS

        -h, --help
                   Show this message and exit.

        -v, --version
                   Display the version information.

FEATURES

        create
                   Create deployments for any application.

        scale
                   Scale the number of jails for a given deployment.

        expose
                   Expose a deployment or service to the external network.

        describe
                   Display detailed information about a deployment, pod, or service.

        logs
                   View the logs of any jail.

        events
                   View full event history for any deployment.
